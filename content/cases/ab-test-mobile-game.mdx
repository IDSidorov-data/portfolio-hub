## Контекст
Free‑to‑play головоломка. Игроки проходят уровни; «гейт» ограничивает прогресс без оплаты/ожидания.

## Гипотеза / цель
Перенос гейта с 40‑го на 30‑й повысит вовлечённость и удержание **D7**.

## Данные / инпуты
Исторические события по пользователям (CSV → PostgreSQL): версии `gate_30`/`gate_40`, retention D1/D7.

## Подход
- ETL: загрузка CSV → PostgreSQL (Docker) → агрегирующий **SQL**.
- Аналитика: расчёт метрик удержания, **z‑тест** разности долей (`statsmodels`).

## Результат / метрики
- **D1:** +0.59 п.п., незначимо (p≈0.074).
- **D7:** **+0.82 п.п.**, значимо (**p=0.0016**).  
**Рекомендация:** раскатить `gate_30` на 100% и наблюдать пост‑эффект.
