## Контекст
Домашний ассистент для управления ПК голосом и выполнения сложных команд.

## Гипотеза / цель
Гибридная (локально + облако) архитектура даст и отзывчивость, и «интеллект» для сложных сценариев.

## Инпуты
Голос пользователя, контекст экрана (скриншот), настройки `.env`.

## Подход
- 3 контура: локальные шаблоны команд → LLM‑классификация → мультимодальный анализ экрана.
- Цикл **самокоррекции** при неудаче: повторная попытка с учётом ошибки.
- Безопасность: маршрутизация API через VPN.

## Результат
Быстрые простые команды (без сети) + сложные сценарии через Gemini; естественное взаимодействие (**STT/TTS**), возможность прерывания.

## Стек
